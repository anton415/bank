# AGENTS.md – Bank Project

## Review guidelines

- Проект: учебный банк на Java 17 + Spring Boot.
- Фокус ревью: **корректность бизнес-логики банка** (балансы, транзакции, инварианты).
- Проверяй, что:
  - Нет побочных эффектов, нарушающих целостность счёта при ошибках.
  - Все новые публичные методы покрыты тестами (unit/integration).
  - Названия методов и классов отражают банковский домен (account, transaction, ledger и т.п.).
  - Нет очевидных проблем с потокобезопасностью при изменении балансов.
  - Код читаемый: разумный размер методов, понятные имена, без дублирования.
- Если есть серьёзная проблема (P0/P1) — чётко опиши риск и предложи конкретное исправление.
- Пожалуйста, **не придирайся к стилю**, если он не ломает читаемость.
